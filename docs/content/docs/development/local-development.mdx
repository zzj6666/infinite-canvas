# 本地开发

当前主应用由 `web/` 前端和 `server/` API 服务组成，画布、素材和媒体按用户保存到服务端。

## 1. 启动服务端

```bash
cd server
bun install
bun run dev
```

## 2. 启动前端

```bash
cd web
bun install
bun run dev
```

前端默认访问：

```text
http://localhost:3000
```

## 3. 配置模型

管理员打开右上角配置弹窗，填写 `Base URL`、`API Key` 和模型名；配置保存在服务端并由 API 代理调用。

## 4. 启动文档站

如果需要单独调整文档站，在 `docs` 目录执行：

```bash
bun run dev
```

## 常见场景

- 改画布、页面和交互：主要看 `web/`
- 改提示词缓存：主要看 `web/src/services/api/prompts.ts`
- 改文档站内容：主要看 `docs/content/docs/`
